== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or install via Plugins > Add New.
2. Activate the plugin.
3. Go to **Settings > Multi-Channel Chat** to configure options.
4. Enable the channels you want and add custom greetings or reason labels.

== Frequently Asked Questions ==

= Does this use tracking or analytics? =  
No. The plugin is fully privacy-compliant and does not include any analytics or third-party scripts.

= Can I show different services on desktop and mobile? =  
Yes. Email channels are shown only on desktop. WhatsApp is mobile-first. You can toggle each individually.

= What if no services are enabled? =  
The widget will not appear. A warning will be shown in the admin panel.

= Can I use this with Facebook Messenger? =  
Yes. Just enter your Facebook Page username or ID.
